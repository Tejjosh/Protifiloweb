Overview
This is the personal portfolio website of Tej Joshi, a computer engineering student and full-stack developer. The site showcases professional experience, skills, certifications, and notable projects, built with PHP, Tailwind CSS, and JavaScript to deliver a responsive, interactive web presence.

Features
Responsive, mobile-first design with Tailwind CSS

Dynamic project gallery and testimonials

Integrated contact form and downloadable CV

Skills section highlighting programming, engineering, and language proficiencies

Detailed education, certifications, and work experience

Light/Dark theme accents and modern UI interactions

Technologies Used
PHP (server-side logic)

HTML5 & JavaScript

Tailwind CSS (styling and layout)

Font Awesome (icons)

JSON-style PHP arrays for dynamic content rendering

Setup Instructions
Clone the repository:

bash
git clone https://github.com/Tejjosh/Protifiloweb.git
Ensure you have PHP (>=7.4) installed.

Place the project files in your local web server (XAMPP/Apache/public_html).

Adjust personal assets, certificate file paths, and avatar image URLs as required.

Open index.php in your browser.

Content Customization
Update $personal_info, $experience, $education, $certifications, $skills, $projects, and $contact_info arrays in PHP as needed.

Replace avatar and CV assets with your own files.

Modify style and theme in the <head> section using Tailwind custom config.
